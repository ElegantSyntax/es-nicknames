# ES-Nicknames

This is a Stardew Valley Mod to add nicknames for the Player.

## Instructions

- Extract the mod into your Mods folder
- Run the game once, a `nicknames.json` file will be added to the mod's folder

```json
{
  "Generic": ["Firstname Lastname"],
  "Casual": ["Firstname"],
  "Friend": ["Firsthname", "Shortname"],
  "Close": ["Shortname"],
  "Best": ["Shortname", "Bestie", "Babe", "Girly"],
  "Spouse": [
    "Firstname",
    "Shortname",
    "Wife",
    "Love",
    "Lover",
    "Darling",
    "Kitten",
    "Honey",
    "Sweetie",
    "Babe"
  ],
  "Roomate": ["Firstname", "Shortname", "Roomie"],
  "Dating": [
    "Firstname",
    "Shortname",
    "Love",
    "Darling",
    "Kitten",
    "Honey",
    "Sweetie",
    "Babe"
  ]
}
```

Edit these names as you see fit.

### Disclaimer

I have no idea if this mod breaks other mods, and it has not been tested beyond Day 2 of a vanilla experience.

Have fun :)
